PeA11y Composants

UXStars

Composant permettant d'afficher deux images superposées, la seconde remplissant proportionnellement la première.
Ce composant est utilisé en particulier pour afficher un score étoile en pourcentage.

<script src="node_modules/@pea11y/uxstars/dist/pea11y.js"></script>
Nom Description Type Défaut
img-empty Chemin de l'image vide (OBLIGATOIRE). string null
img-full Chemin de l'image remplie (OBLIGATOIRE). string null
img-size Taille de l'image utilisée (OBLIGATOIRE). number[] null
value Valeur sélectionnée. int 0
label Texte affiché à coté de l'image. string null

.pea11y-uxstars-label { 
  margin-left: 15px; 
  vertical-align: text-top; 
} 
.pea11y-uxstars-empty { vertical-align: middle; }
function init() { 
  var elemUxstars = document.getElementsByTagName("pea11y-uxstars"); 
  for (var i = 0, l = elemUxstars.length; i < l; i++) { 
    elemUxstars[i].imgSize = [78, 15]; 
  } 
} 
function resetUxstars(idElem) { 
  document.getElementById(idElem).setAttribute("value", 0); 
}
