PeA11y Composants accessibles

Notation

Composant de type potentiomètre permettant de choisir un score avec les flèches ou à la souris.

Le composant demande la librairie JQuery 3.3.1 minimum. 

<link rel="stylesheet" href="pe-a11y/rating/rating.css">
<script src="pe-a11y/rating/rating.js"></script>

Il est obligatoire de renseigner soit un label, soit un labelledBy, sinon le composant sera en erreur pour des raisons d'accessibilité.

Nom Description Type Défaut
max Nombre d'icones. number 5
initValue Valeur initiale sélectionnée du composant. number 0
allowEmpty Autorise ou non 0 comme valeur sélectionnable. boolean true
readOnly Mode lecture seule. boolean false
textValue Donne des alternatives textuelles aux valeurs numériques. string[] null
labelledBy Donne l'id de l'élément HTML qui sert de label au composant. string null
label Donne un label textuel au composant rendu uniquement par les lecteurs d'écran. string null
templateFull Template HTML de l'icone remplie. string "★"
templateEmpty Template HTML de l'icone vide. string "☆"
