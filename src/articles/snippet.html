Snippet

Composant permettant d'afficher du code dans plusieurs langages, sous forme d'onglets et avec une coloration syntaxique.
Il a besoin de la librairie PrismJS pour fonctionner, et il faut obligatoirement lier un thème prismjs.

Nom Description Type Défaut
data-html Chemin du morceau de code HTML. string null
data-js Chemin du morceau de code Javascript. string null
data-css Chemin du morceau de code CSS. string null
