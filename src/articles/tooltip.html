Tooltip

Composant permettant d'afficher une information au focus sur un élément.

Nom Description Type Défaut
tooltip-label Chaîne contenant le texte à afficher (OBLIGATOIRE). string null
same-as-txt Mettre à vrai si le texte de l'élément est le même que celui du tooltip afin d'éviter des doublons au lecteur d'écran. boolean false

Compétence en boulangerie
