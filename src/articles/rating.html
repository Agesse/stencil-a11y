Notation

Composant de type potentiomètre permettant de choisir un score avec les flèches ou à la souris.
Il est obligatoire de renseigner soit l'attribut label, soit l'attribut labelledBy, sinon le composant sera en erreur pour des raisons d'accessibilité.

Nom Description Type Défaut
max Nombre d'icones affichées. number 5
value Valeur sélectionnée a l'initialisation du composant. number 0
allow-empty Autorise ou non 0 comme valeur sélectionnable. boolean true
readOnly Mode lecture seule. boolean false
text-values Donne des alternatives textuelles aux valeurs numériques. string[] null
labelledby Donne l'id de l'élément HTML qui sert de label au composant. string null
label Donne un label textuel au composant rendu uniquement par les lecteurs d'écran. string null
template-selected Template HTML appliqué de la 1ère icône à l'icône dont la valeur est sélectionnée. string "★"
template-empty Template HTML appliqué aux icônes après la valeur sélectionnée. string "☆"
template-hover Template HTML appliqué de la 1ère icône à l'icône survolée. string "★"
