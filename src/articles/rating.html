Notation

Composant permettant de choisir un score avec les flèches ou à la souris.
Il est obligatoire de renseigner soit l'attribut label, soit l'attribut labelledby, sinon le composant sera en erreur pour des raisons d'accessibilité.

Nom Description Type Défaut
max Nombre d'icones affichées. number 5
value Valeur sélectionnée a l'initialisation du composant. number 0
readonly Mode lecture seule. boolean false
text-values Donne des alternatives textuelles aux valeurs numériques. string[] null
labelledby Donne l'id de l'élément HTML qui sert de label au composant. string null
label Donne un label textuel au composant rendu uniquement par les lecteurs d'écran. string null
class-selected Classes CSS appliquées de la 1ère icône à l'icône dont la valeur est sélectionnée. string "fas fa-star"
class-empty Classes CSS appliquées aux icônes après la valeur sélectionnée. string "far fa-star"
class-hover Classes CSS appliquées de la 1ère icône à l'icône survolée. string "fas fa-star"
Nom Description Retour
@onChange Evènement déclenché lors d’un changement de la valeur sélectionnée.
Envoie la valeur sélectionnée.		 int
@onMouseOver Evènement déclenché lors d’un changement de la valeur survolée.
Envoie la valeur survolée.		 int
